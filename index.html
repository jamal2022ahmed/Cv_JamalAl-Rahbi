كيفية نشر السيرة الذاتية على الإنترنت

  1. إنشاء حساب على GitHub (إذا لم يكن لديك حساب) من خلال هذا الرابط
  2. إنشاء مستودع جديد باسم my-cv أو أي اسم تفضله
  3. رفع الملفات إلى المستودع عن طريق سحب وإسقاط ملف HTML أو استخدام الأوامر
  4. تفعيل GitHub Pages من خلال الذهاب إلى إعدادات المستودع (Settings) → Pages → اختيار فرع main → حفظ
  5. انتظر بضع دقائق ثم انتقل إلى العنوان https://اسمك.github.io/my-cv لمشاهدة سيرتك الذاتية على الإنترنت

بدائل أخرى للنشر: Netlify أو Vercel وتقدم خصائص إضافية ومجانية أيضًا.
